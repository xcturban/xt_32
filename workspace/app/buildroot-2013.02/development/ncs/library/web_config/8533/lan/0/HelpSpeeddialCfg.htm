Calling mode：
1.Single-button mode：The terminal can only call one setting target via emergency button.

2.Dual-button mode：The terminal can call two diffent targets via emergency button and the virtual button on the screen.

3.Contact list mode：Contact list has two display mode,Notice:Please do not put more than 4 pictures since the terminal memory is limited.
a.Graphic：The terminal can display customized graphics corresponding to the graphics' name, 1.png is corresponding to the 1st contact list page, 2.png is correspongind to the 2nd page...etc. The user can visit terminal via FTP(the user name and password all is "admin") to acquire the pattern picture contact_list_template.png, put the customized graphics or let the terminal download graphics from the FTP server.
b.Text：The terminal will display the setting target name by text when the corresponding page graphics is not exist.
Notice:Please do not put more than 4 pictures since the terminal memory is limited
3.1Dispaly column：Each page can display 5 rows, the user can choose to display 1 or 2 column, that is, each page can display 5 or 10 calling targets.
3.2Name：Display the calling target name.
3.3Phone：The calling target number.

4.Soft keyboard mode：The user can call any target number using the soft keyborad.
4.1Open the door password：The user can enter the password to trigger general purpose output port to open the door.
